Look up product_id for this guid/version Create a new record in products if it doesn't exist
Only pass through each product_id to the rest of the transformation once. This step does not queue rows. It uses an in memory hashset to determine uniqueness.
Check whether this product_id has already had the additional data updated in it. (version details, product name)
There are a static set of guids in the products table that contain canonical names for each product. This query retrieves those for use in the update.
The stream at this point contains the product_id, product name, and version details (major/minor/sub_a/sub_b). Update the product_id with this additional data.
