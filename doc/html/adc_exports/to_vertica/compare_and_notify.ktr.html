This transformation compares mysql and vertica - same table - same product and reports differences: changes or deleted data. For running alone, the following variables need to be set: PRODUCT_ID (id from d_firefox_mobileproduct) TABLE_NAME (for ex. f_firefox_language_installs) KEY_FIELD (for comparing, this depends on the table name for ex. language_id)
