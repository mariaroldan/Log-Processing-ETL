Look up location_id based on geohash Create a new record in "locations" if it doesn't exist
Only pass through each location_id to the rest of the transformation once. This step does not queue rows. It uses an in memory hashset to determine uniqueness.
Check whether this location_id has already had the additional data updated in it.
The stream at this point contains the location_id, continent, country, region, city, lat/long. Update the location_id with this additional data.
